Rep Calculator for Pace-Based Running Workouts
-----------------

A simple single-page tool for running coaches to calculate the right number of reps per pace group, so every group finishes a workout at the same time.
Built for Tuesday night track/interval sessions where runners are grouped by pace rather than experience level. Set your workout window, rep distance, and recovery, then dial in the rep count for your fastest group — the calculator scales reps for all other groups automatically.

Features

* Anchor-based rep scaling: set reps for Group A, all others are calculated to match finish time
* Recovery by time (MM:SS) or distance (with per-group pace adjustment)
* Configurable pace groups with custom names and ranges
* Printable coaching sheet
* No install, no dependencies — open the HTML file in any browser
